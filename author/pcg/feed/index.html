]]> get-dhcp-opts is a tool to discover DHCP/BOOTP servers on your LAN, and dump the DHCP/BOOTP options.

Sometimes network infrastructures use DHCP/BOOTP to provide special configurations. For example, a VOIP network can use these special options to configure the phones (VoIP server address, configuration file URLs, …).

get-dhcp-opts display these options and detect rogue DHCP servers on your network.
